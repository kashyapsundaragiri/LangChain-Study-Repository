# LangChain Study Repository

This repository contains my hands-on learning journey with LangChain, Retrieval-Augmented Generation (RAG), vector databases, embeddings, and LLM application development.

## Topics Covered

* Data Ingestion
* Text Splitting Techniques
* Embedding Fundamentals
* HuggingFace Embeddings
* Ollama Embeddings
* ChromaDB Vector Store
* FAISS Vector Store
* OpenAI Integration
* LangSmith Tracing
* Retrieval-Augmented Generation (RAG)

## Project Structure

```text
01_Data_Ingestion.ipynb
02_Text_Splitter.ipynb
03_Embedding_Techniques.ipynb
04_HuggingFace_Embeddings.ipynb
05_Ollama_Embeddings.ipynb
06_ChromaDB.ipynb
07_FAISS.ipynb
OpenAI/
```

## Setup

```bash
pip install -r requirements.txt
```

Create a `.env` file and add:

```env
OPENAI_API_KEY=your_key_here
LANGCHAIN_API_KEY=your_key_here
LANGCHAIN_PROJECT=your_project_name
```

This repository is intended as a personal learning resource and practical reference while studying modern LLM application development and RAG systems.
